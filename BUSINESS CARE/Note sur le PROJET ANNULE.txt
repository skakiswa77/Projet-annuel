Note sur le PROJET ANNULE
Pour réaliser la mission 1 on dois suivre cela:
Créer une application web avec plusieurs espaces:
-Espace pour les sociétés clientes(gestion des contrats, factures,collaborateurs)
-Espace pour les salairiés(réservation de services,gestion du planning,chatbot)
-Espace pour les prestataires(gestion des interventions,calendrier,facturation)
-Back-office d'administration générale(gestion complète de l'activité de     Business Care).
Fonctionnalités clés :
   Gestion des contrats et des paiements (via Stripe).
   Génération automatique de factures en PDF.
   Notifications push pour les salariés.
   Site multilingue (sans utiliser Google Translate).
   Utilisation d'une API pour gérer les traitements.
   Serveur web personnel pour héberger le site.
